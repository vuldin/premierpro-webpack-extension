## (Starting point for) an extension for Adobe Premier Pro

Uses yarn, babel, webpack, react

# Install

```bash
npm i -g yarn
yarn
```

# Run locally

```bash
yarn start
```

# Build

```bash
yarn run build
```

# Export to Premier Pro

```bash
cp -a CSXS lib dist index.html /path/to/{premier-pro-extension-folder}/new-plugin-folder
```
